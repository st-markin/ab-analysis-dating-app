## **Всем привет!**

Данный проект посвящён анализу A/B-эксперимента по изменению стоимости премиум-подписки в дейтинговом приложении.

Работа выполнена на **Python** с использованием **Jupyter Notebook**.

<hr>

### **Задача**   
Представьте, что вы работаете в крупном дейтинговом приложении.  

Помимо базовых функций, в приложении также имеется премиум-подписка, которая дает доступ к ряду важных дополнительных возможностей. Был проведен A/B тест, в рамках которого для новых пользователей из нескольких стран была изменена стоимость премиум-подписки* при покупке через две новые платежные системы. При этом стоимость пробного периода оставалась прежней.  

Необходимо проверить:  

Был ли эксперимент успешен в целом?  

*Деньги за подписку списываются ежемесячно до тех пор, пока пользователь её не отменит.  

<hr>

### **Реализация проекта:**
* Использован API для загрузки датасетов.  
* Проведён предварительный анализ данных (EDA) и их предобработка.  
* Выполнена проверка логической корректности данных и устранение выявленных ошибок.  
* Определены ключевые метрики анализа: Conversion Rate, ARPU и ARPPU.  
* Проведена проверка групп control_1 и control_2 на отсутствие статистически значимых различий по ключевым параметрам с использованием статистических тестов в Python (U-тест Манна—Уитни, Z-тест для разности пропорций, χ²-тест).  
* Выполнен анализ данных в разрезе стран.  
* Проведена проверка групп control и test на отсутствие статистически значимых различий по ключевым параметрам с использованием статистических тестов в Python (U-тест Манна—Уитни, χ²-тест).  
* Рассчитаны метрики Conversion Rate, ARPU и ARPPU для каждой группы.  
* Проведена проверка статистической значимости различий между группами по рассчитанным метрикам с использованием статистических методов в Python (bootstrap-подход, Z-тест для разности пропорций).  
* Выполнена фильтрация выбросов с использованием метода межквартильного размаха (IQR) и повторный расчёт метрик.  
* Визуализированы распределения данных с использованием Python-библиотек seaborn и matplotlib.
